---
name: status-report
description: Post a What I Did / What's Left / What Needs Your Call update to Slack.
---

When asked for a status update, gather: tasks completed since the last update, tasks still remaining, and any decision or blocker that requires human input.

Reply in exactly three sections:

**What I Did**
List each completed task as a short bullet. Be specific — include file names, endpoints, or feature names where relevant.

**What's Left**
List remaining tasks in the order you plan to tackle them.

**What Needs Your Call**
List any decisions, approvals, or blockers that require human input before you can proceed. If there are none, write "Nothing — proceeding autonomously."

Do not add extra sections, preamble, or sign-offs. Keep each bullet under 15 words.
